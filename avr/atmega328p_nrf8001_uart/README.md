# Atmega328p nrf8001  

A small project to interface an adafruit nrf8001 BTLE module with an atmega328p 

## Required HW
- 1 Atmega328P (target)
- 1 Arduino Uno/Nano (ISP)
- 1 nrf8001 Adafruit Breakout 

## Required SW
- ArduinoISP Sketch
- avr toolchain (Installed automatically with Arduino IDE):
  - avr-gcc 
  - avr-dude
  - avr-objcopy
