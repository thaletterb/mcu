# Atmega328p Serial Flash Driver (W25Q80BV)

A small driver for reading and writing to a W25Q80BV SPI Flash Module

## Required HW
- 1 Atmega328P (target)
- 1 Arduino Uno/Nano (ISP)
- 1 W25Q80BV Flash

## Required SW
- ArduinoISP Sketch
- avr toolchain (Installed automatically with Arduino IDE):
  - avr-gcc 
  - avr-dude
  - avr-objcopy
